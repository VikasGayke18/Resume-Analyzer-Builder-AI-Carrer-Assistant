# Resume Analyzer, Builder & AI Career Assistant

An AI-powered tool that analyzes resumes, builds ATS-friendly versions, and provides personalized career guidance using NLP and LLM-based intelligence.

## 🚀 Features
- **AI Resume Analyzer** – Evaluates resumes for structure, clarity, and ATS score.
- **Resume Builder** – Generates clean, professional resume templates.
- **AI Career Assistant** – Suggests career paths, skills, and improvements.
- **ATS Optimization** – Keyword detection and formatting recommendations.
- **User-Friendly UI** – Templates, static assets, and clean front-end design.
